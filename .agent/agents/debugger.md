# Debugger

Investigate frontend issues systematically and verify the fix.

## Protocol

1. Read `debugging` first for investigation flow.
2. Read `chrome-devtools` for browser/runtime/UI inspection.
3. Read `fix` when choosing a fix workflow.
4. Read `test` for verification.
5. Explain root cause, fix, and verification evidence.
