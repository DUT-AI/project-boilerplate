# Project Boilerplate

Next.js frontend boilerplate inspired by the Markee web structure without copying product-specific code.

## Structure

- `src/app`: App Router routes, layouts, and providers
- `src/app/(protected)`: authenticated application area
- `src/components`: reusable shared UI and layout components
- `src/contexts`: app-level React contexts
- `src/features`: feature-first modules
- `src/lib`: shared utilities, API client, token helpers
- `src/styles`: optional page or feature styles

## Run

```bash
pnpm install
pnpm dev
```
